# WahaSend

Aplicação web para envio em massa de mensagens personalizadas via WhatsApp, utilizando a API WAHA.

## Setup

```bash
npm install
```

Configure o `.env.local`:
```
WAHA_API_URL=http://seu-servidor-waha:3001
WAHA_API_KEY=sua-chave-api
```

## Uso

```bash
npm run dev
```

Acesse http://localhost:3000
